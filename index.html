STYLE TESTER, Git Diffs for CSS
  • Designed and developed a test framework to check the client-level impact of stylesheet changes, as opposed to the textual changes highlighted by Git Diffs
  • The tool runs as part of the Maps Stylesheet CI pipeline, compiles the stylesheets (pre and post changes) and replicates iOS client behavior to obtain the client level impact.
  • This information is then visualized in a web browser locally, which provides detailed insight into the changes being merged. Optionally, the users can choose to upload/persist the diff to a remote server (Orchard).
  • The tool operates prior to PRs being merged and dramatically increases the odds of preventing stylesheet regressions.
  • Tech Stack: Objective-C (Compilation tool for stylesheets and client-behavior replication), Python/Flask (Main controller and Flask hosts the diff on localhost), d3.js (visualizing the diff data) and Orchard (Heroku-based, for persisting diffs remotely)
TARGETED IMAGE ANALYSIS (TIA), Identifying Display Impact Automatically
  • Visually testing every tile on the map is impossible. TIA automatically identifies the tiles that are impacted by Stylesheet changes.
  • It's a fully automated pipeline which does the following things:
    • Identifies the stylesheet changes using the STYLE TESTER module
    • Runs Map Reduce jobs on close to 500GB worth of the world's tile data, stored in HDFS
    • Creates a mapping between style attributes and the tiles containing them.
    • Identifies locations impacted by stylesheet changes, visualizes them in an internal image comparison tool and sends email reports as and when comparisons are ready for display evaluation.
  • The tool runs as part of every server side release and helps reduce the scope of display testing without needing to arbitrarily pick locations to analyze.
  • Tech Stack: Python (Main Controller, API Calls, Email Reports), Objective-C (Identifying Stylesheet changes at the client level), Spectrum (Internal tool for running Map Reduce Jobs on HDFS. Jobs are written in Groovy)
VECTOR STATS, Providing Real Time Metrics/KPIs on nightly CI builds
